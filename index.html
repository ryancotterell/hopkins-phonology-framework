Welcome to the Hopkins Phonology Framework.

The Hopkins Phonology Framework provides a toolkit for building string-valued graphical models for modeling phonology and inferring underlying forms of words. The framework is build on OpenFST and offers an implementation of several inference algorithms as well as pre-built models. We also release datasets we adapted from exercises in phonology textbooks.
