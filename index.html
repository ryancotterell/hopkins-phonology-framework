Welcome to the Hopkins Phonology Framework.

The Hopkins Phonology Framework provides a toolkit for building string-valued graphical models for modeling phonology and inferring underlying forms of words. The framework is built on OpenFST and offers an implementation of several inference algorithms as well as pre-built models. We also release datasets we adapted from exercises in phonology textbooks.

Code

Prerequisites

I've augmented the pyfst library by Victor Chahuneau with some additional features. I added Cython headers to avoid going through Python, first order expectation semirings, signed log space, graph search for expected n-gram count extraction and a few other minor things. I plan to wrap the OpenGRM library as well as the push-down transducer extension in the near future.

Instructions

  • install OpenFST 1.4.1
  • pip install cython
  • git clone https://github.com/ryancotterell/hopkins-phonology-framework.git
  • install mustache
  • cd pyfst
  • make
